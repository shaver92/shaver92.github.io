3.3 定义训练层

上面需要了解下轮次和批次的概念

如上整个截图（文本的所有特征分布）是所有的数据集的文本条数，红框部分是定义的训练批次，这里大模型能够学习到整个文本的特征就像是盲人摸象

训练批次就是盲人一次能够摸到的面积，如果盲人手足够大，一次能够摸完，那就可以正确的做出结论来，显然这是不可能的，同理我们的训练批次也是越大越好，但是越大，越占用GPU内存，所以得在机器运行范围内设置的尽量大

训练轮次是指训练完所有数据集中的数据一次为一轮，这里可以设置的足够大，后面我们的模型训练达到交付标准提前终止即可

4.1 观察状态

我们在训练的过程中，需要观察模型训练的趋势（精度和损失），需要尽量保证模型达到拟合的状态，这里介绍下模型的三个状态（欠拟合，拟合，过拟合），如下图所示

欠拟合：模型的分布弱于数据的真实分布 

**拟合：**模型的分布恰巧表达数据的分布  理想状态 

**过拟合：**模型过度拟合数据的分布规律 会是模型的结果过度依赖数据的噪声信息，一旦数据的场景发生细微的变化就回导致结果错误

为防止过拟合（过拟合是不可逆转的过程，所以我们训练的时候需要保存好对应过拟合之前的参数）

#模型训练
import torch
from data_loader import MyDataset
from torch.utils.data import DataLoader
from half_net import Model
from transformers import BertTokenizer,AdamW
#定义设备信息
DEVICE = torch.device("cuda" if torch.cuda.is_available() else "cpu")
#加载字典和分词器
token = BertTokenizer.from_pretrained(
    r"/Users/shaver/PycharmProjects/ai_model_learn/02_bert_train_test/model/bert-base-chinese/models--bert-base-chinese/snapshots/c30a6ed22ab4564dc1e3b2ecbf6e766b0611a33f")
#将传入的字符串进行编码
def collate_fn(data):
    sents = [i[0]for i in data]
    label = [i[1] for i in data]
    #编码
    data = token.batch_encode_plus(
        batch_text_or_text_pairs=sents,
        # 当句子长度大于max_length(上限是model_max_length)时，截断
        truncation=True,
        max_length=512,
        # 一律补0到max_length
        padding="max_length",
        # 可取值为tf,pt,np,默认为list
        return_tensors="pt",
        # 返回序列长度
        return_length=True
    )
    input_ids = data["input_ids"]
    attention_mask = data["attention_mask"]
    token_type_ids = data["token_type_ids"]
    label = torch.LongTensor(label)
    return input_ids,attention_mask,token_type_ids,label
#创建数据集
test_dataset = MyDataset("test")
test_loader = DataLoader(
    dataset=test_dataset,
    #训练批次
    batch_size=100,
    #打乱数据集
    shuffle=True,
    #舍弃最后一个批次的数据，防止形状出错
    drop_last=True,
    #对加载的数据进行编码
    collate_fn=collate_fn
)
if __name__ == '__main__':
    acc = 0.0
    total = 0
    #开始测试
    print(DEVICE)
    model = Model().to(DEVICE)
    #加载模型训练参数
    model.load_state_dict(torch.load("params/16_bert.pth"))
    #开启测试模式
    model.eval()
    for i,(input_ids,attention_mask,token_type_ids,label) in enumerate(test_loader):
        #将数据放到DVEVICE上面
        input_ids, attention_mask, token_type_ids, label = input_ids.to(DEVICE),attention_mask.to(DEVICE),token_type_ids.to(DEVICE),label.to(DEVICE)
        #前向计算（将数据输入模型得到输出）
        out = model(input_ids,attention_mask,token_type_ids)
        out = out.argmax(dim=1)
        acc += (out==label).sum().item()
        print(i,(out==label).sum().item())
        total+=len(label)
    print(f"test acc:{acc/total}")```

# 5 AI大模型的三大指标

 a.**精度**
    模型的推理准确度

**b.性能**
    模型的推理速度，上面的精度越高，在同样的机器下，性能就会越低

**c.泛化性**
   模型在经过训练后，一般对于训练的数据会有比较好的响应，但是对于没有经过训练的数据是否表现的出色就在于模型的泛化性，一般欠拟合和过拟合的模型的泛化性都比较差，欠拟合一般比较好理解，那过拟合呢，过拟合一样的，因为过拟合的模型会受到数据噪声的印象，比如训练的数据是猫的图片，每个猫的图片中都有一个毛线球，后面再给一张没有毛线球的猫的图片，模型不能识别这是猫的图片
