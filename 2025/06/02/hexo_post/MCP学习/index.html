MCP 学习

1 基本了解

MCP（Model Context Protocol 模型上下文协议）是Anthropic开源的一种开放协议，可以实现LLM应用程序与外部数据源和工具无缝集成

可以理解为以前我们需要在本地LLM到工具调用的整个链路都需要实现，现在只需要按照协议标准在本地连接上服务端即可，服务的实现可以单独拿出进行开发
